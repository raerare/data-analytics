## Project Overview

This project analyzes sales data from the Iowa Liquor Sales database, focusing on tequila. The goal is to understand how tequila is performing across stores, vendors, and counties, and to identify trends, strengths, and opportunities using SQL and data visualization.

---

## Project Requirements Completed

### 1. SQL Script

I created one SQL script that answers all 20 capstone required analysis questions.
The queries include:

* Filtering and sorting sales data
* Aggregating revenue and averages
* Joining tables (sales, stores, products, counties)
* Using subqueries to find advanced insights

The script analyzes:

* Sales trends over time
* Top products and vendors
* High-performing stores
* Revenue by county
* Market opportunities

---

### 2. Data Visualization

I generated insights using toools like PostgreSQL (pgAdmin), Tableau, and Microsoft Excel (PivotTable & PivotChart) to support my analysis

---

## Key Insights from Data

* A small number of vendors dominate tequila sales
* A few products generate most of the revenue ("hero" products)
* Sales are highest in certain counties, especially Polk County
* Some stores perform much better than others
* Tequila sales show some seasonal trends
* Mid-priced tequila products tend to sell the most
